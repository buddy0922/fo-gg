# FO.GG – FC온라인 전적 분석 웹 서비스

## 📌 프로젝트 목표
FC온라인 유저들에게 OP.GG처럼 전적 분석, 플레이 스타일 분석, 메타 추천, 전술 추천 기능을 제공하는 웹 서비스입니다.

## 🧩 MVP 기능
- 전적 검색
- 유저 프로필 조회 (티어, 기록)
- 경기 종류별 최근 경기 확인 (스페셜/공식/친선)
- 플레이 스타일 자동 분석
- 메인 페이지: 전적 검색 + 메타 추천 메뉴

## 🛠 기술 스택(예정)
- Frontend: React + TypeScript
- Backend: Node.js (Express)
- Infra: Vercel or AWS
- Database: PostgreSQL / Supabase

## 📡 Data Source
Nexon FC Online Open API
